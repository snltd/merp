## Little Tests

These exercise doers (mostly) in isolation.

They pretty much all need root privileges to work, and a lot of them must be
run in the global zone.

Wherever it's sensible, the test actions are loaded from
[the Gurp config examples](https://github.com/snltd/gurp/tree/main/janet/examples),
so you need a Gurp config.

Edit `site.janet` for your network and whatnot.
